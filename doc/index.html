Installing the Template

Thank you for buying from me. This is a simple how-to guide to help you successfully install your new template.

Educa template is for education. All files are well organized and layered for proffesionals and agencies working in the construction. It has a lot of features and varianty to use, it is built on top of bootstrap 3 the template offers a creative solution with stunning design and code best practices in mind, we've done our best from the first line of code to the last one, we grant functionality across all modern browsers and mobile devices with readable fonts and retina graphics.

01) Uploading to Server

Since this is a HTML template, no extra configuration is required. All you will have to do is upload the template to your server and enjoy. First, open the folder you have unzipped to your favorite location. Please save your work at this point or make a backup before trying to install.

Upload the contents of folder "HTML Files" (not the folder "HTML Files" itself) to your FTP root directory on your server location with your FTP software.

If you don't like this color scheme which is provided and want to use your colors, you can simply costumize with less by editin variables on variables.less file

If you do not have the FTP Client software, you can get one here for free: http://filezilla-project.org

After upload, navigate with your browser to the template installation folder. Template visuals will appear on your screen as shown in the demo. Installation is now complete.

Editing and Adding Content

01) Bootstrap Framework

This template is built entirely on top of Bootstrap, Twitter HTML framework. It has many great reusable features such as: fluid design, tables, buttons, javascript components, menus, etc.

For complete and extensive documentation on core operation of this template, we recommend to look at the Bootstraps web documentation, available online here: http://getbootstrap.com/

02) Changing the Color Scheme

We've built this template with last trends of LESS language so you can simply change variables of colors or fonts on the file assets/educa-less/variables.less.

03) Replacing the Logo

To replace a logo, go to the line 49 and replace image of logo, size 157x142

04) Font Awesome Icons

Font Awesome (http://fortawesome.github.io/Font-Awesome/icons/) provides incredible Retina ready & javascript powered icons. To include an icon to your html page copy & paste this example code:

            <i class="fa fa-facebook"></i>

The above code will display the Facebook icon. You can find over 369 icons on Font Awesome webpage to include anywhere in your web page.

CSS Files and Structure

This section covers all styling files you will need in case of doing larger modifications to the template. This template is built for fast deployment and no conflict operation. All template code is located in a separated files from the Bootstrap core engine.

01) Cascading Style Sheets

Main style files are located in folder "assets/css". FontAwesome sources is located in folder "assets/font-awesome".

  1. animate.css - is a bunch of cool, fun, and cross-browser animations that we used on our template.
  2. bootstrap.min.css - grid columns and all other main things that build layout included here.
  3. educa.css - include style of everything, like padding, margin font-sizes.
  4. font-awesome.min.css - this files comes from FontAwesome Icons. Here are provided class you can use.

JavaScript

Javascript files are located in folder /js/

We have added the following javascript files and folder:

  1. custom.js - Main webpage JavaScript, Plugins Init, Site Specific Functions, ShortCodes etc...
  2. jquery-1.10.2.min.js - jQuery is a fast, small, and feature-rich JavaScript library.
  3. bootstrap.min.js - bootstrap javascript files with their functions.
  4. plugins.js - i've included all js plugins used on template in this file.
  5. jquery.themepunch.revolution.min.js - Revolution Slider 4.x Scripts
  6. jquery.themepunch.tools.min.js - Revolution Slider 4.x Scripts
